# GB Oracle Skills

Oracle não é só escrever SELECT.

É entender versão, contexto, RAC, Data Guard, impacto em produção, cardinalidade e plano de execução.

Esse repositório organiza conhecimento real de campo para agentes de IA escreverem SQL e procedimentos Oracle com padrão de DBA sênior.

Sem mágica.
Sem chute.
Com responsabilidade.

---

## O que é isso?

Um conjunto estruturado de:

- SKILL.md → mapa de decisão
- references/ → conhecimento aprofundado por tema

O objetivo é guiar qualquer agente a pensar como DBA Oracle de produção.
